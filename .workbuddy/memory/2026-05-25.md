## 快捷键扩展方案（JSON驱动）
- 用户要求：只在 server.js 中加配置，不影响原有 keyMap 逻辑，不需要额外文件
- 方案：在 keyMap 之后新增 `shortcutConfig` JSON 对象 + `execShortcut()` 通用执行函数
- 原有 keyMap 及所有 if/else 分支完全不动
- 非JSON路径末尾追加 `else if (shortcutConfig[cmdStr])` 兜底
- JSON路径末尾追加 `else if (shortcutConfig[type])` 兜底
- 支持五种类型：普通按键、组合键(modifiers)、鼠标(mouse)、打开(open)、连招(sequence)
- 预置快捷键：ctrl_w, ctrl_pgup, ctrl_pgdn, ctrl_t, ctrl_shift_t, ctrl_s
- 新增快捷键只需在 shortcutConfig 中加一行即可，无需改动执行逻辑
